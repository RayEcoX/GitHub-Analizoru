Projenin Anatomisi

Amacımız, teknolojimiz ve geliştirme sürecimiz.

Misyon

Modern Web Pratiği

Bu proje; sadece arama yapan standart bir araç değil. Vanilla JavaScript ile DOM ağacını inşa etmeye, veri tiplerini anlamaya ve Fetch API üzerinden gelen JSON datasını tasarıma yedirmeye adanmış bir laboratuvardır.

Teknoloji

Sıfır bağımlılık (No-dependency). HTML5, Vanilla ES6 JavaScript ve Bootstrap 5 Grid sistemi.

UI / UX Mimarisi

Minimalist Monokrom tasarımla dikkat dağıtmayan, tam odaklı Bento Box yaklaşımı ve pürüzsüz Stagger animasyonları.

API Entegrasyonu

GitHub Public REST API üzerinden milyonlarca geliştiricinin istatistikleri, monokrom kartlarda render edilmek üzere anlık çekilmektedir.

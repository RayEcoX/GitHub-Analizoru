# 🔍 GitHub Profil Analizörü

Herhangi bir GitHub kullanıcısının profilini analiz edebileceğiniz, modern ve minimal tasarımlı bir web uygulaması.

## ✨ Özellikler

- **GitHub API Entegrasyonu** — Gerçek zamanlı kullanıcı profili ve repo verileri
- **Profil Analizi** — Takipçi, repo sayısı, katılım tarihi ve daha fazlası
- **Top Repolar** — Yıldız sıralamasına göre en popüler 6 repo
- **Dil Renk Haritası** — Her programlama dili için GitHub standart renkleri
- **Dark / Light Mode** — LocalStorage ile kalıcı tema tercihi
- **Arama Geçmişi** — Son aramaların chip'ler ile kolay erişimi
- **SweetAlert2** — Modern ve şık bildirim/uyarı animasyonları
- **Tamamen Responsive** — Mobil, tablet ve masaüstü uyumlu

## 🛠️ Teknolojiler

| Teknoloji | Kullanım |
|-----------|----------|
| HTML5 | Yapısal iskelet |
| CSS3 | Custom Properties, animasyonlar, Bento Box grid |
| JavaScript (ES6+) | Fetch API, Async/Await, DOM manipülasyonu |
| Bootstrap 5.3 | Responsive grid, bileşenler |
| SweetAlert2 | Bildirim ve uyarı popup'ları |
| GitHub REST API | Kullanıcı ve repo verisi |

## 💻 Yerel Kurulum

Projeyi klonlayın ve herhangi bir tarayıcıda açın:

```bash
git clone https://github.com/RayEcoX/GitHub-Analizoru.git
cd GitHub-Analizoru
# index.html dosyasını tarayıcınızda açın
```

> Herhangi bir build adımı veya bağımlılık yüklemeye gerek yoktur. Statik dosyaları doğrudan açabilirsiniz.
